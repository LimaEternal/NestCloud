NestCloud - простое облачное хранилище файлов

Данное веб-приложение для хранения и управления файлами написано на Flask, база данных на SQLAlchemy, аутентификация пользователей реализована с помощью Flask-Login, обработка изображений для превью выполняется с использованием Pillow, формы созданы на базе Flask-WTF. Проект доступен в GitHub-репозитории.
