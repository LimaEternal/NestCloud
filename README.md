# NestCloud - простое облачное хранилище файлов

## Общая информация
Веб-приложение для хранения и управления файлами написано на [Flask](https://flask.palletsprojects.com/), база данных на [SQLAlchemy](https://www.sqlalchemy.org/), аутентификация пользователей реализована с помощью [Flask-Login](https://flask-login.readthedocs.io/), обработка изображений для превью выполняется с использованием [Pillow](https://pillow.readthedocs.io/), формы созданы на базе [Flask-WTF](https://flask-wtf.readthedocs.io/).


## Функционал приложения

### Аутентификация:
- Регистрация новых пользователей
- Вход в систему
- Выход из аккаунта
- Защита маршрутов от неавторизованного доступа

### Управление файлами:
- Загрузка файлов (максимальный размер — 512 МБ)
- Просмотр списка загруженных файлов
- Скачивание файлов с оригинальными именами
- Удаление файлов (включая удаление превью и записей из базы данных)
- Переименование файлов (изменение имени без изменения расширения)

### Превью и визуализация:
- Автоматическая генерация превью для изображений (PNG, JPG, JPEG, WEBP, BMP, GIF)
- Отображение иконок для различных типов файлов:
  - Архивы (ZIP, RAR, 7Z, TAR, GZ, BZ2)
  - Аудио (MP3, WAV, OGG, FLAC, AAC, M4A)
  - Видео (MP4, AVI, MOV, MKV, WMV, FLV)
  - Документы (PDF, DOC, DOCX, TXT, RTF, XLS, XLSX, PPT, PPTX)
  - Другие файлы

### Интерфейс:
- Сортировка файлов по имени, размеру и времени загрузки
- Отображение размера файлов в человекочитаемом формате (B, KB, MB, GB)
- Адаптивный дизайн с использованием Bootstrap
- Информативные сообщения об успешных операциях и ошибках

### Безопасность:
- Хеширование паролей с использованием Werkzeug
- Изоляция файлов пользователей (каждый пользователь имеет свою папку)
- Проверка прав доступа при операциях с файлами
- Валидация загружаемых файлов
- Защита от недопустимых символов в именах файлов


## Запуск приложения
Создание виртуального окружения:
```bash
python -m venv venv
```
Активация виртуального окружения:
```bash
./venv/Scripts/activate
```
Установка необходимых модулей:
```bash
pip install flask flask-sqlalchemy flask-login flask-wtf python-dotenv pillow werkzeug
```
Для запуска приложения вам необходимо поместить файл `.env` в директорию проекта со следующим содержимым:
```
SECRET_KEY=your_secret_key_here
DATABASE=sqlite:///instance/cloud.db
```
Запуск приложения:
```bash
python app.py
```
